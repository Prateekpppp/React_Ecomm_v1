# Pre-requisites
- Nodejs
- npm
- mySql
- mongoDB Client using vscode extension or mongo compass


# Getting started
- Clone the repository
```
git clone  https://github.com/Prateekpppp/React_Ecomm_v1.git
```
- Install dependencies
```
cd React_Ecomm_v1
npm install
```
- Build and run the project
```
npm run dev
```
  Navigate to `http://localhost:5173`
